# Yu-Yu Chen's Portfolio

Welcome to my personal portfolio! This website showcases my work in **Data Science, Machine Learning, and Psychometrics**, along with my **presentations, projects, and skills**.

## 🚀 Live Website
Check out the deployed version:  [**ycyukichen.github.io**](https://ycyukichen.github.io/yuyu.github.io/)

---

## 📌 Features
- **Home**: Introduction to my work and expertise.
- **Work**: Overview of my core skills in Data Science, AI, and Psychometrics.
- **Skills**: A categorized list of technologies and tools I work with.
- **Presentations**: Academic and professional talks I have given.
- **Articles & Publications**: Sharing insights on Data Science, AI, and Psychometrics.
- **Portfolio**: My latest projects with links to GitHub repositories.
- **Contact**: A form to reach out for collaborations.
